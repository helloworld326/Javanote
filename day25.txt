事件监听机制：
	1.事件源	时间发生的地方
	2.事件		要发生的事情
	3.事件处理	针对发生的事情做出的处理方案
	4.事件监听	把事件源和事件关联起来
	
	举例：人受伤
		事件源：	人(具体对象)
			Person p1 = new Person("张三");
			Person p2 = new Person("李四");
		事件：受伤
			interface 受伤接口{
				一拳();
				一脚();
				一板砖();
			}
		时间处理：
			受伤处理类 implements 受伤接口{
				一拳(){}
				一脚(){}
				一板砖(){}
			}
		事件监听：
			p1.注册监听(受伤接口);
			
局部内部类访问局部变量要用final修饰;
